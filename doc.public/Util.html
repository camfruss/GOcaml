val string_of_list : ('a -> string) -> 'a list -> string

string_of_list cast lst concatenates all the elements in lst to a single string, with each element cast to a string according to cast.

val max_triple3 : (int * int * int) list -> int * int * int

max_triple3 lst finds the largest positive third element of a triple in lst and returns the entire triple. Requires all third elements be natural numbers.

val quartet_swap : ('a * 'b * 'c * 'd) array -> ('b * 'a * 'd * 'c) array

quartet_swap a swaps the 1st with the 2nd and the 3rd with the 4th element in each quartet.

val cartesian : 'a list -> 'b list -> ('a * 'b) list

cartesian lst1 lst2 takes the cartesian product of lst1 with lst2.

val time : unit -> int

time is the current Unix time as an integer.

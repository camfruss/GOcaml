Module Game

module Game: sig .. end

Game contains the logic required for loading, saving, and playing a GOcaml. It represents the internal game state and does not provide any user-facing interactive capabilities.

type t

t is the abstract type of values representing go games.

type stone =
| Black
| White

stone is the type representing the black or white stones

type intersection =
| WhiteT
| BlackT
| Neutral
| WhiteS
| BlackS
| Empty

intersection is the type representing how a single intersection may be viewed when scoring a Go game. Example: White stone and white territory correspond to WhiteS and WhiteT, respectively.

exception KoException

KoException is raised when the configuration of a Go board is repeated twice in a single game.

exception SelfCaptureException

SelfCaptureException is raised when a player attempt to place a stone in a position that is immediately captured.

exception StoneAlreadyExistsException

StoneAlreadyExists is raised when a player attempts to place a stone on top of an existing stone.

exception TimeExpiredException

TimeExpiredException is raised when a player has exhausted their entire game clock.

exception GameEndException

GameEndException is raised when two players consecutively pass which signals the game is over

val from_json : Yojson.Basic.t -> t

from_json json is the Go game that json represents. Requires: json is a valid json Go representation.

val to_json : t -> string -> unit

to_json go str writes the contents of go into a valid json. Does not overwrite pre-existing files..

val in_bounds : t -> int * int -> bool

in_bounds t (c,r) is whether the stone at column c and row r is within the bounds of the board.

val bounds : t -> int

bounds t is the nxn size of the board in game t.

val is_empty : t -> int * int -> bool

is_empty t pos is whether there is no stone current at position pos in the t.

val score : t -> float * float

score t is the current score of the game using the territory scoring. The first value in the tuple represents the score of the black stones and the second value, the white stones.

val liberties : t -> int * int -> int

liberties t pos is the number of liberties in the group at input pos. Requires: a stone is currently placed at pos.

val ko : t -> int * int -> bool

ko t str is whether an input str would violate ko.

val stones : t -> stone -> (int * int) list

stones t s is the col and row of every stone on the board of type stone.

val turn : t -> stone

turn t is the stone of the player whose turn it currently is.

val names : t -> string * string

names t is the pair of player names, with the first value playing with black stones and the second with white.

val star_locations : t -> int * int * int

star_locations t are the three possible locations for a star point on a Go board. Star points act as a reference for Go boards and mark where handicap stones, if any, are placed.

val handicap_c : t -> int -> (int * int) list

handicap_c t h is the positions of h handicap stones on the board in t.

val handicap : t -> (int * int) list -> t

handicap t lst places the handicap stones specified in lst on the board. It then becomes white's turn.

val last_stone : t -> int * int

last_stone t is the column and row, respecively, of the last stone placed on the board in t. Returns: (-1, -1) if there are no stones on the board.

val step : t -> (int * int) option -> int -> t

step t move time is the new game after the current player places a stone on move and spends time seconds doing so.

val string_of_board : t -> string

string_of_board t is the string representation of a board.

val fill_grid : t -> intersection array array

fill_grid t is the board in t with black, neutral, and blwhiteack territory mark accordingly, as well as black and white stones.

val undo : t -> t

undo t reverts to the game at beginning of the previous player's turn excluding time.

val game_message : t -> string list * string list * string list

game_message t describes all the relevant information about t corresponding to the stat name, player 1 value, and player 2 value.
